# Legacy-Themes (ältere Themes)

Sehr alte Themes unterstützen mitunter keine **App-Blocks** bzw. App-Embeds. In diesem Fall binden Sie den Zugang zum Widerrufsformular über einen **fertigen Button-Code** oder einen **Menü-Link** ein.

::: steps
1. **Widerrufsseite anlegen**
   Legen Sie zunächst in den **Formular-Einstellungen** die Widerrufsseite an (siehe [Inline-Formular](./inline-formular.md)).

2. **Button-Code generieren**
   Öffnen Sie in der App die **Button-Einstellungen**. Dort erzeugt Dotcase einen fertigen Code-Schnipsel, den Sie in den HTML-/Code-Bereich einer Seite einfügen können.

3. **Code einfügen**
   Fügen Sie den Schnipsel in der gewünschten Seite über den **HTML-/Code-Editor** Ihres Themes oder einer Shop-Seite ein.

4. **Alternativ: Menü-Link**
   Verlinken Sie die Widerrufsseite über **Inhalte → Menüs → Footer-Menü** als normalen Menüpunkt.
:::

::: callout warning "In die HTML-Ansicht einfügen" icon:code
Wenn Ihr Editor zwischen Text- und HTML-Ansicht unterscheidet, fügen Sie den Code in die **HTML-/Code-Ansicht** ein (Symbol `<>`). In der reinen Textansicht würde der Code als Text angezeigt statt als Button.
:::

::: callout tip "Modernes Theme?" icon:info
Nutzt Ihr Shop ein modernes Theme (Online Store 2.0), verwenden Sie einfacher das [App-Embed](./app-embed-button.md) – ganz ohne Code.
:::
