# App-Embed-Button

Das App-Embed **„EU Withdrawal Button"** zeigt einen **schwebenden Button** auf den Storefront-Seiten Ihres Shops an. Ein Klick öffnet das Widerrufsformular. Sie aktivieren es einmalig im Theme-Editor – ohne Eingriff in den Theme-Code.

::: steps
1. **Theme-Editor öffnen**
   Shopify-Admin → **Onlineshop → Themes → Anpassen**. Alternativ öffnet die App über den Einrichtungsassistenten den Editor direkt.

2. **App-Embeds aufrufen**
   Klicken Sie unten links auf das **App-Embeds**-Symbol (Steckersymbol).

3. **„EU Withdrawal Button" einschalten**
   Aktivieren Sie den Schalter für das App-Embed von Dotcase.

4. **Speichern**
   Speichern Sie das Theme. Der Button erscheint nun auf der Storefront.
:::

::: callout tip "Aussehen anpassen" icon:palette
Den **Stil des Absenden-Buttons** (Hintergrund, Textfarbe, Eckenradius, Ausrichtung) und die Formularfelder verwalten Sie in den **Formular-Einstellungen** der App (Button-Stil ist eine Pro-Funktion) – nicht im Theme-Editor.
:::

::: callout warning "App-Embed aktiviert lassen" icon:alert-triangle
Damit Button und Formular funktionieren, muss das App-Embed dauerhaft eingeschaltet bleiben. Ist es deaktiviert, erscheint der Button nicht.
:::

::: callout info "Reicht der Button allein?" icon:info
Der Button leitet zum Widerrufsformular. Wenn Sie zusätzlich eine **eigene Widerrufsseite** wünschen, sehen Sie unter [Inline-Formular](./inline-formular.md).
:::
