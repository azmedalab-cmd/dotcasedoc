# Formular einbinden

Damit Ihre Kundschaft den Widerruf ausüben kann, brauchen Sie zwei Dinge im Shop:

1. eine **Widerrufsseite** mit dem eigentlichen **Formular** und/oder
2. einen **Button/Link**, über den die Kundschaft zum Formular gelangt.

Wählen Sie den Weg, der zu Ihrem Theme passt:

::: grids
  ::: grid
    ::: card "App-Embed-Button" icon:mouse-pointer-click
    Ein schwebender Button auf allen Storefront-Seiten, aktiviert im Theme-Editor.

    [Anleitung →](./app-embed-button.md)
    :::
  :::
  ::: grid
    ::: card "Inline-Formular" icon:file-text
    Das Formular fest auf einer eigenen Widerrufsseite.

    [Anleitung →](./inline-formular.md)
    :::
  :::
:::

::: grids
  ::: grid
    ::: card "Legacy-Themes" icon:history
    Für ältere Themes ohne App-Blocks.

    [Anleitung →](./legacy-themes.md)
    :::
  :::
  ::: grid
    ::: card "Link als Button" icon:link
    Footer-/Menü-Link oder die Rückgaberichtlinie als Button gestalten.

    [Anleitung →](./link-als-button.md)
    :::
  :::
:::

::: callout tip "Aussehen des Buttons" icon:palette
Felder, die Widerrufsseite und der **Stil des Absenden-Buttons** (Hintergrund, Textfarbe, Eckenradius, Ausrichtung) werden zentral in den **Formular-Einstellungen** der App verwaltet (Button-Stil ist eine Pro-Funktion). Sie müssen am Theme-Code nichts ändern.
:::

::: callout warning "Platzierung ist Ihre Verantwortung" icon:alert-triangle
Der Widerrufsbutton muss **leicht zugänglich und gut sichtbar** sein. Prüfen Sie die Platzierung in Ihrem Live-Shop selbst.
:::
