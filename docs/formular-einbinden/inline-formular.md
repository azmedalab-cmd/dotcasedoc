# Inline-Formular auf einer Widerrufsseite

Dotcase kann eine eigenständige **Widerrufsseite** in Ihrem Shop anlegen, auf der das Widerrufsformular **fest eingebettet** angezeigt wird. Das ist ideal, wenn Sie der Kundschaft eine feste Adresse (z. B. im Footer) nennen möchten.

::: steps
1. **Formular-Einstellungen öffnen**
   In der App: **Formular-Einstellungen**.

2. **„Widerrufsseite anlegen" klicken**
   Dotcase erstellt automatisch eine eigene Shop-Seite, die das Formular enthält. Die Seite wird einmalig erstellt und danach wiederverwendet.

3. **App-Embed aktiv lassen**
   Damit das Formular auf der Seite gerendert wird, muss das App-Embed **„EU Withdrawal Button"** im Theme-Editor aktiviert sein (siehe [App-Embed-Button](./app-embed-button.md)).

4. **Seite verlinken**
   Verlinken Sie die neue Widerrufsseite z. B. im Footer-Menü Ihres Shops oder als Button (siehe [Link als Button](./link-als-button.md)).
:::

::: callout tip "Formularfelder festlegen" icon:sliders
Welche Felder Pflicht sind und welche Texte erscheinen, stellen Sie in den **Formular-Einstellungen** ein. Die **Teilwiderruf-Funktion** (Auswahl einzelner Produkte) ist eine Pro-Funktion.
:::

::: callout warning "Seite versehentlich gelöscht?" icon:alert-triangle
Wurde die Widerrufsseite entfernt, legen Sie sie einfach erneut über **„Widerrufsseite anlegen"** an.
:::
