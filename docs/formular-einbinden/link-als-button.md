# Link als Button

Sie entscheiden, **wie** Ihre Kundschaft zum Widerrufsformular gelangt. Dotcase bietet dafür mehrere Wege, die Sie in den **Button-Einstellungen** der App konfigurieren.

## Möglichkeiten

::: card "1. Seite ins Footer-Menü aufnehmen" icon:menu
**Inhalte → Menüs → Footer-Menü → Menüpunkt hinzufügen** und die Widerrufsseite auswählen. Der einfachste Weg.
:::

::: card "2. Button auf einer beliebigen Seite" icon:square-mouse-pointer
Lassen Sie sich in den **Button-Einstellungen** einen Button-Code erzeugen und fügen Sie ihn in die HTML-Ansicht der gewünschten Seite ein.
:::

::: card "3. Button auf der Rückgaberichtlinie-Seite" icon:file-badge
Generieren Sie einen Button für Ihre Shopify-Richtlinienseite (Rückgaberichtlinie). Da Richtlinienseiten nicht automatisch bearbeitet werden können, liefert die App einen fertigen Code-Schnipsel samt Schritt-für-Schritt-Anleitung.
:::

::: steps
1. **Button-Einstellungen öffnen**
   In der App: **Button-Einstellungen**.

2. **Weg wählen und Code/Link erzeugen**
   Wählen Sie eine der Möglichkeiten oben und übernehmen Sie den erzeugten Link bzw. Code.

3. **Im Shop einfügen und prüfen**
   Fügen Sie ihn an der gewünschten Stelle ein und prüfen Sie im Live-Shop, ob der Button korrekt erscheint und zum Formular führt.
:::

::: callout warning "Richtlinienseite: HTML-Editor nutzen" icon:code
Fügen Sie den Code in die **HTML-Ansicht** (`<>`) ein – nicht in den reinen Texteditor – sonst erscheint der Button nicht.
:::

::: callout info "Label der Order-Status-Schaltfläche" icon:tag
Den Text der Widerruf-Schaltfläche auf der **Bestellstatus-Seite** (Order-Status, Pro) passen Sie ebenfalls in den Button-Einstellungen an – je Sprache.
:::
