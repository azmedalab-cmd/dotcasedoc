# Datenschutz & AVV

## Was ist der AVV?

Wenn Dotcase personenbezogene Daten Ihrer Kundschaft verarbeitet, geschieht das **in Ihrem Auftrag**. Sie sind der **Verantwortliche**, Dotcase ist **Auftragsverarbeiter**. Gemäß **Art. 28 DSGVO** ist dafür ein **Auftragsverarbeitungsvertrag (AVV)** erforderlich.

Der AVV von Dotcase basiert auf den **EU-Standardvertragsklauseln für Auftragsverarbeiter** (Durchführungsbeschluss (EU) 2021/915). Die verbindliche Fassung ist **auf Deutsch**.

## AVV ansehen & herunterladen

::: steps
1. **Im Gate (Schritt 0)**
   Bei der ersten Nutzung akzeptieren Sie den AVV. Über **„AVV ansehen"** öffnen Sie das Dokument jederzeit.

2. **Personalisiertes PDF**
   Nach der Annahme können Sie über **„AVV als PDF herunterladen"** ein auf Ihre Firma personalisiertes, datiertes AVV-Dokument speichern (Ihre in Schritt 0 bestätigten Angaben stehen in Anlage I).
:::

::: callout info "AGB" icon:file-text
Neben dem AVV akzeptieren Sie die **AGB** (Allgemeine Geschäftsbedingungen). Auch diese sind auf Deutsch und im Gate einsehbar.
:::

Für die optionalen Standort-Funktionen wird die Besucher-IP **lokal** (eingebettete GeoLite2-Datenbank) und **nur im Arbeitsspeicher** ausgewertet – sie wird **nicht** an Dritte übertragen und **nicht** gespeichert.
:::

## Wie lange werden Daten gespeichert?

Abgeschlossene Widerrufsanträge werden **12 Monate** nach Abschluss **anonymisiert** (nicht mehr personenbeziehbar). Nach Deinstallation werden die Auftragsdaten innerhalb von **30 Tagen gelöscht**. Details: [Einstellungen](./einstellungen.md).

::: callout warning "Keine Rechtsberatung" icon:alert-triangle
Diese Seite fasst die Datenverarbeitung zusammen und ersetzt keine Rechtsberatung. Maßgeblich sind der AVV, die AGB und die Datenschutzerklärung in ihrer jeweils gültigen Fassung.
:::
