# Pro & Abrechnung

Dotcase gibt es in einem dauerhaft kostenlosen **Free**-Tarif und einem **Pro**-Tarif mit erweiterten Funktionen.

## Free vs Pro

| Funktion | Free | Pro |
|---|:---:|:---:|
| Unbegrenzte Widerrufsanträge | ✓ | ✓ |
| 24 EU-Sprachen im Formular | ✓ | ✓ |
| Widerrufsseite & Formular | ✓ | ✓ |
| Dashboard (annehmen / ablehnen / löschen) | ✓ | ✓ |
| Kunden-Bestätigung & Händler-Benachrichtigung | ✓ | ✓ |
| **Gewährleistungs-Label** (EU-Hinweis) | ✓ | ✓ |
| Genehmigungs-/Ablehnungs-E-Mails | – | ✓ |
| Editierbare E-Mail-Vorlagen | – | ✓ |
| Button-/Formular-Styling | – | ✓ |
| Teilwiderruf (Produktauswahl) | – | ✓ |
| CSV-Export | – | ✓ |
| PDF-Export je Antrag | – | ✓ |
| Order-Status-Extension (Bestellstatus-Seite) | – | ✓ |
| Shopify-Returns-Synchronisierung | – | ✓ |
| Konfigurierbare Widerrufsfrist(-Anzeige) | – | ✓ |

::: callout success "Gewährleistungs-Label immer kostenlos" icon:gift
Der EU-Gewährleistungshinweis ist auf **allen** Tarifen kostenlos – auch im Free-Tarif.
:::

## Preise

::: callout info "Aktuelle Preise im App Store" icon:tag
Pro kostet derzeit **9,90 USD / Monat** bzw. **99 USD / Jahr**, jeweils mit einer **14-tägigen kostenlosen Testphase**. **Maßgeblich sind stets die im Shopify App Store ausgewiesenen, aktuellen Preise.**
:::

## Abrechnung über Shopify

::: steps
1. **Auf Pro upgraden**
   Das Upgrade läuft über die von Shopify gehostete Abrechnungsseite (Shopify Billing). Die App führt Sie dorthin.

2. **Testphase nutzen**
   Während der 14-tägigen Testphase fallen keine Kosten an.

3. **Kündigen**
   Sie können jederzeit zum Ende des laufenden Abrechnungszeitraums kündigen – für Sie geschieht das durch die reguläre **Deinstallation** der App.
:::

::: callout tip "Rechnung & Zahlung" icon:receipt
Zahlungsabwicklung, Rechnungsstellung und Inkasso erfolgen ausschließlich über **Shopify** (Shopify Billing), in der Regel in US-Dollar. Es gelten die [AGB](./datenschutz-avv.md).
:::
