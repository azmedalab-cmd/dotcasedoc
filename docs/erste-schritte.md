# Erste Schritte

In rund **5 Minuten** ist Dotcase einsatzbereit. Folgen Sie diesen vier Schritten.

::: steps
1. **App installieren**
   Installieren Sie **Dotcase – EU Compliance** aus dem Shopify App Store. Sie werden dabei einmalig durch die Berechtigungen geführt. Details: [Installation & Gate](./installation.md).

2. **AVV & AGB akzeptieren (Schritt 0)**
   Beim ersten Öffnen prüfen und bestätigen Sie Ihre Verantwortlichen-Daten (Firma, Anschrift) und akzeptieren den **Auftragsverarbeitungsvertrag (AVV)** und die **AGB**. Anschließend können Sie Ihren personalisierten AVV als PDF herunterladen.

3. **Button bzw. Formular einbinden**
   Aktivieren Sie im **Theme-Editor** das App-Embed **„EU Withdrawal Button"** und/oder legen Sie in den **Formular-Einstellungen** eine **Widerrufsseite** an. Siehe [Formular einbinden](./formular-einbinden/index.md).

4. **E-Mails prüfen**
   Senden Sie einen Test-Widerruf ab und prüfen Sie die Bestätigungs-E-Mail an die Kundschaft sowie die Benachrichtigung an Sie. Optional passen Sie die Texte an (Pro): [E-Mail-Vorlagen](./email-vorlagen.md).
:::

::: callout tip "Einrichtungsassistent im Admin" icon:wand
Im Dashboard führt Sie ein **7-stufiger Einrichtungsassistent** durch genau diese Schritte – inklusive direkter Links zum Theme-Editor und zu den Einstellungen.
:::

::: callout warning "Ihre Verantwortung" icon:alert-triangle
Der Widerrufsbutton muss in Ihrem Theme **leicht zugänglich und gut sichtbar** platziert sein. Prüfen Sie nach dem Einbinden im Live-Shop selbst, ob der Button erscheint und das Formular funktioniert.
:::

## Wie geht es weiter?

- Den Button verfeinern: [App-Embed-Button](./formular-einbinden/app-embed-button.md)
- Den kostenlosen EU-Hinweis anzeigen: [Gewährleistungs-Label](./gewaehrleistungs-label.md)
- Eingehende Widerrufe verwalten: das **Dashboard** im Shopify-Admin (Annehmen / Ablehnen / Löschen).
