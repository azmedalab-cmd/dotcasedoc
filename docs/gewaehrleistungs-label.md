# Gewährleistungs-Label (Richtlinie (EU) 2024/825)

Dotcase kann den **amtlichen Hinweis auf die gesetzliche Gewährleistung** in Ihrem Shop anzeigen. Grundlage ist die **Richtlinie (EU) 2024/825**. Angezeigt wird die offizielle, von der EU bereitgestellte Grafik – unverändert und in der passenden Sprache.

::: callout success "Auf allen Tarifen kostenlos" icon:gift
Das Gewährleistungs-Label ist **immer kostenlos** – auch im Free-Tarif.
:::

## Aktivieren

Das Label ist ein **Theme-App-Block** und wird im Theme-Editor platziert.

::: steps
1. **Compliance-Labels öffnen**
   In der App: **Compliance-Labels**. Dort sehen Sie eine Vorschau in Ihrer Admin-Sprache und einen Link zum Theme-Editor.

2. **Block im Theme hinzufügen**
   Fügen Sie im Theme-Editor den Dotcase-Block für den Gewährleistungshinweis an der gewünschten Stelle hinzu (z. B. auf der Produktseite oder im Footer).

3. **Speichern & prüfen**
   Speichern Sie das Theme und prüfen Sie die Anzeige im Live-Shop.
:::

::: callout info "Optional – bald verpflichtend" icon:calendar
Der Hinweis ist derzeit **optional** und wird zu einem von der EU vorgegebenen Stichtag verpflichtend. Die App weist Sie im Einrichtungsassistenten auf den jeweiligen Stichtag hin. [TEYİT: exaktes Pflicht-Datum / Geltungsbereich bitte bestätigen]
:::

::: callout tip "Mehrsprachig" icon:languages
Die Grafik wird automatisch in der passenden Sprache angezeigt. Sie müssen keine Texte selbst pflegen.
:::
